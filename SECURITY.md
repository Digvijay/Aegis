## Reporting a Vulnerability

We take the security of Aegis very seriously. If you have discovered a security vulnerability, please do NOT create a public GitHub issue.

Instead, please report it privately via email to **security@aegis-protocol.io**.

We will review the issue and respond within 48 hours.
